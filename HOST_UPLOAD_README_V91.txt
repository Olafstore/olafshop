OLAF SHOP — HOST UPLOAD V91
Date: 2026-07-13

สิ่งที่เปลี่ยนจาก V90
- เอาแถบเมนูร้านค้า/หมวดหมู่/ค้นหาของ UI ใหม่ออก
- เอาส่วน “เลือกดูสินค้าเพิ่มเติม” ออก
- เอาพื้นหลัง gradient ที่ครอบทับพื้นหลังเว็บไซต์เดิมออก
- ปรับสินค้าเด่นและแนะนำให้เต็มความกว้างและมีความสูงสม่ำเสมอ
- เพิ่มคอลเลกชันสินค้าแนวภาพใหญ่/ภาพย่อยต่อจากส่วนดีล
- เพิ่มแถวแนะนำสินค้าแบบ Hero + Gallery โดยใช้รูปและสินค้าในร้านจริง
- ปรับแถวสินค้าแนะนำใน product.html บนมือถือให้มีระยะห่าง เห็นการ์ดถัดไป และปัดได้
- รองรับ Mobile, Tablet และ Desktop

ไฟล์ที่เปลี่ยน
- index.html
- app.js
- styles.css

วิธีอัปโหลด
1. สำรองไฟล์บน Host ปัจจุบัน
2. อัปโหลดเนื้อหาทั้งหมดภายในโฟลเดอร์นี้ไปยัง Document Root
3. เลือก overwrite/replace ไฟล์เดิม
4. รอ Deploy เสร็จ
5. เปิด Incognito หรือ Hard Refresh เพื่อตรวจ cache version V91

ตรวจหลังอัปโหลด
- ไม่พบแถบหมวดหมู่สีน้ำเงินของ UI V90
- ไม่พบหัวข้อ “เลือกดูสินค้าเพิ่มเติม”
- Hero สินค้าเด่นเต็มความกว้างและรูปไม่ยืด
- คอลเลกชันสินค้าใหม่แสดงต่อจาก “ดีลและสินค้าน่าสนใจ”
- Mobile ปัดคอลเลกชันและ Gallery ได้ทีละการ์ด
- หน้า product.html เห็นขอบการ์ดแนะนำใบถัดไปและมีช่องว่างระหว่างการ์ด

Database / SQL
- ไม่ต้องรัน SQL
- ไม่มีการแก้ schema, RPC, stock, order หรือ payment flow

Security
- ไม่มี credentials หรือ service-role key ฝังใน Browser bundle
- ใช้ Product ID และข้อมูลแค็ตตาล็อกเดิมของ OLAF SHOP
