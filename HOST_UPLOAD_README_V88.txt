OLAF SHOP - HOST UPLOAD V88
วันที่จัดทำ: 2026-07-13

สิ่งที่เปลี่ยนในชุดนี้
1. สินค้า Active ที่สต็อกเป็น 0 ยังคงแสดงในหน้าเว็บทุกหมวดหมู่
2. สินค้าสต็อก 0 แสดงสถานะ "สินค้าหมด" และถูกจัดไว้ท้ายรายการโดยอัตโนมัติ
3. เมื่อเติมสต็อก สินค้าจะกลับไปยังตำแหน่งเดิมตามลำดับสินค้า/ตัวเลือกการเรียง
4. ตัวกรอง "เฉพาะพร้อมส่ง" ยังใช้งานได้ แต่ค่าเริ่มต้นจะไม่เปิด เพื่อไม่ซ่อนสินค้าหมด
5. หน้าเติม Point ปรับธีมใหม่ พร้อม responsive และ animation สำหรับมือถือ
6. ปรับ animation เปิด/ปิดหน้าต่าง QR ให้สมูทและล็อกตำแหน่งบนมือถือ

วิธีอัปโหลด
- อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดภายในโฟลเดอร์นี้ไปทับไฟล์เดิมบน Host
- คงโครงสร้างโฟลเดอร์เดิม โดยเฉพาะ api และ assets
- ล้าง cache ของ Host/CDN หรือ Redeploy หลังอัปโหลด
- ทดสอบหน้า index.html, more-products.html และ point-topup.html

ฐานข้อมูล
- ชุด V88 นี้ไม่ต้องรัน SQL และไม่แก้ข้อมูลสินค้าในฐานข้อมูล
- ระบบใช้สถานะ Active และค่า stock ปัจจุบันจากฐานข้อมูลเดิม
