OLAF SHOP - HOST UPLOAD V89
วันที่จัดทำ: 2026-07-13

รายการปรับปรุง
1. เร่งโหลดรูปการ์ดหน้า Index โดยให้รูปแถวที่กำลังมองเห็นได้ priority ก่อน
2. รูปที่ใกล้ viewport จะเริ่มโหลดล่วงหน้าอัตโนมัติประมาณ 720px
3. กำหนดขนาดรูป 16:9 ล่วงหน้าเพื่อลด layout shift
4. โลโก้ Desktop, iPad, Mobile และเมนูมือถือใช้ค่าไอคอนเว็บจากหน้า Admin
5. ปรับการ์ดสินค้าแนะนำบนมือถือให้กว้างพอดี ไม่มีช่องว่างเกินจำเป็น
6. ปรับ touch animation ของการ์ดบนมือถือให้กระชับและลื่นขึ้น
7. รวมการแก้ไข V88 เรื่องสินค้าหมดเรียงท้ายและหน้าเติม Point ไว้แล้ว

วิธีอัปโหลด
- แตก ZIP แล้วอัปโหลดไฟล์และโฟลเดอร์ทั้งหมดไปทับไฟล์เดิมบน Host
- รักษาโครงสร้างโฟลเดอร์ api และ assets
- ล้าง Cache/CDN หรือ Redeploy หลังอัปโหลด
- ตั้งโลโก้จาก Admin > การตั้งค่าร้าน > ไอคอนเว็บ แล้วกดบันทึก

ฐานข้อมูล
- ไม่ต้องรัน SQL
- ไม่แก้ข้อมูลสินค้า สต็อก หรือคำสั่งซื้อ

ผลทดสอบ
- Phase 5-12 regression: 42 ผ่าน, 0 ล้มเหลว
- Slip / Point / Security regression: 20 ผ่าน, 0 ล้มเหลว
- JavaScript syntax ผ่าน: app.js, image-performance.js, site-navigation.js

ความปลอดภัย
- ไม่มี .env, Service Role Key, RDCW Secret หรือไฟล์บัญชี Offline ในชุดอัปโหลด
