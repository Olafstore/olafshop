OLAF SHOP — HOST UPLOAD V93
วันที่จัดแพ็กเกจ: 2026-07-13

สิ่งที่ปรับปรุง
1. ล็อกป๊อปอัป QR ชำระเงินให้อยู่กึ่งกลางทุกขนาดจอ ไม่ยืดเต็มจอและไม่กระโดดเมื่อเปิด/ปิด
2. ใช้แอนิเมชันกับการ์ดด้านใน ลดการกระพริบและไม่เปลี่ยนตำแหน่ง dialog
3. ปรับหน้าเติม Point พร้อมตัวอย่างยอดแบบสด การ์ดข้อมูล และเอฟเฟกต์เปลี่ยนยอด
4. เพิ่มป๊อปอัปกิจกรรมหน้า Index รองรับภาพเดสก์ท็อป 1080x1920 และภาพมือถือ 1:1
5. ป๊อปอัปกิจกรรมมีปุ่มปิด ปุ่มไม่แสดง 24 ชั่วโมง ลิงก์กิจกรรม และพื้นที่เลื่อน
6. หน้าแอดมินมีเมนู “กิจกรรมหน้าเว็บ” สำหรับเปิด/ปิด อัปโหลดภาพ ใส่ข้อความ และแนบลิงก์

วิธีอัปโหลด
1. สำรองไฟล์บนโฮสก่อนทุกครั้ง
2. อัปโหลดไฟล์และโฟลเดอร์ทั้งหมดในโฟลเดอร์นี้ไปยัง Document Root ของเว็บ
3. เลือก Overwrite/Replace ไฟล์ชื่อเดิม และรักษาโครงสร้างโฟลเดอร์ api และ assets
4. ถ้าใช้ Vercel/GitHub ให้อัปโหลดเนื้อหาในโฟลเดอร์นี้เข้า root ของ repository (ไม่อัปโหลดโฟลเดอร์ครอบซ้ำอีกชั้น)
5. รอ deployment เสร็จ แล้ว Hard Refresh หรือเปิด Incognito เพื่อตรวจ cache รุ่น 20260713-activity-payment-v93

ตั้งค่ากิจกรรม
1. เข้า olaf-control.html ด้วยบัญชีแอดมิน
2. เปิดเมนู “กิจกรรมหน้าเว็บ”
3. กำหนดรหัสกิจกรรม ชื่อ ข้อความ ลิงก์ ป้ายปุ่ม และอัปโหลดภาพ Desktop/Mobile
4. เปิดสถานะกิจกรรมแล้วกดบันทึก
5. เมื่อต้องการให้กิจกรรมใหม่แสดงแก่คนที่เคยกดไม่แสดง 24 ชั่วโมง ให้เปลี่ยน “รหัสกิจกรรม”

ฐานข้อมูล
- ไม่ต้องรัน SQL เพิ่ม รุ่นนี้ใช้ store_settings และ Storage bucket payment-qr ที่ระบบมีอยู่แล้ว
- ไม่มีการฝัง Service Role หรือข้อมูลลับในไฟล์ frontend

การตรวจสอบ
- JavaScript syntax ผ่าน 5 ไฟล์หลัก
- Automated tests ผ่าน 66 รายการ
- CSS braces สมดุล
- ตรวจภาพ QR มือถือ 390x844 และป๊อปอัปกิจกรรมทั้งมือถือ/เดสก์ท็อปแล้ว

หมายเหตุ
- ยังไม่ได้ Deploy, Push GitHub หรือรัน SQL Production จากชุดงานนี้
